- "Smoke test" came from hardware engineering, after assembling electronic circuit boards, 
   literally look for smoke.

- Perfomed with minimial load to make sure the system is working.

- Helps establish a baseline before running expensive load tests

Virtual Users: 1-2
Duration:      (1-5) mins
When to run:   Immediately when we get a new build
               It stimulates 1-2 users browsing a webpage

                                                Typical K6 config:
                                                ------------------

export let options = {
    vus: 2,         // 2 Virtual Users
    duration: '1m'  // Run for 1 min
}
